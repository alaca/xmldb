A) Introduction - top

XML Db is a flat-file database that use XML files for storing and organizing data. It doesn't need any database systems such as MySQL or SQLite.

B) Xml database structure - top

<?xml version="1.0" encoding="utf-8"?>
<database>
    <table>
        <row>
            <id>1</id>
            <column1>one</column1>
            <column2>two</column2>
            <column3>three</column3>
        </row>
        <row>
            <id>2</id>
            <column1>four</column1>
            <column2>five</column2>
            <column3>six</column3>
        </row>
        ...
    </table>
    <another_table>
        <row>
            <id>1</id>
            <another_column1>one</another_column1>
            <another_column2>two</another_column2>
            <another_column3>three</another_column3>
        </row>   
        <row>
            <id>2</id>
            <another_column1>four</another_column1>
            <another_column2>five</another_column2>
            <another_column3>six</another_column3>
        </row>  
        ...
    </another_table>
</database>

C) API - top

  • xmlDb::conect($database) connect to database
  • xmlDb::deleteDatabase($database) delete database
  • xmlDb::chmodDatabase($database, $permissions) change permissions of database file
  • xmlDb::getDatabasePerms($database) get permissions of database file
  • backup() create database backup
  • restore() restore database backup
  • addTable($name) add table to database
  • removeTable($name) remove table from database
  • getTables() get table names from database
  • addColumn($name, $value) add column and column value to table
  • removeColumn($name) remove column from table
  • getColumns($table) get column names from table
  • insert($data) insert data into database table
  • update($data) update data from database table
  • delete() delete data from database
  • getRow() get one row
  • getAll() get all rows
  • table($name) select table
  • join($table, $primary_key, $foreign_key) join table on primary key and foreign key
  • select($columns) list of columns to select
  • where($query) xpath query
  • limit($num) limit result
  • affectedRows() num of rows affected
  • bind($placholder, $value) bind data
  • orderBy($column, $direction) order result by column direction
  • lastId() get last insert id from database

D) Using XML xmlDb - top

Connecting to database - top
// include xmlDb class
require 'database/xmldb.php';

$xmlDb = xmlDb::connect('database');

// variable $xmlDb is now database object instance
If database file not exists it will be created automatically. Database files are stored in data directory which is protected with htaccess file.

Delete database - top
// include xmlDb class
require 'database/xmldb.php';

// delete database
xmlDb::deleteDatabase('database');
Delete the database file from data directory.

Chmod database - top
// include xmlDb class
require 'database/xmldb.php';

// chmod database file
xmlDb::chmodDatabase('database', 0755);
Change database file permissions. Default permissions are set to 0644. See http://www.php.net/chmod for more info.

Get database file permissions - top
// include xmlDb class
require 'database/xmldb.php';

echo 'Database permissions ' . xmlDb::getDatabasePerms('database');

Add column - top
// include xmlDb class
require 'database/xmldb.php';

// connect to database 
$xmlDb = xmlDb::connect('database');

// add table
$xmlDb->addTable('example_table');

// for doing anything with table (adding columns, removing columns, selecting data) 
// first we must select table by calling method table(table_name)

// add column "id" and set its value to 1
$xmlDb->table('example_table')->addColumn('id', 1);

// add another column called "name"
$xmlDb->table('example_table')->addColumn('name', 'John');

// add another column called "lastname"
$xmlDb->table('example_table')->addColumn('lastname', 'Doe');
addColumn() method is intended to extend the table structure, but can be used when creating new table for adding columns.
If you add columns in new table on this way, column id must be set.

Better way for adding columns in newly created table is by using method insert()

Insert data - top
// include xmlDb class
require 'database/xmldb.php';

// connect to database 
$xmlDb = xmlDb::connect('database');

// you can insert data in two different ways

// by passing an array with column names and their values to insert method
$data = array(
    'name'     => 'My name',
    'lastname' => 'My last name'
);

$xmlDb->table('example_table')->insert($data);


// or by using method bind(column_name, value) with insert
$xmlDb->table('example_table')
      ->bind('name', 'My other name') 
      ->bind('lastname', 'My other last name')
      ->insert();


// example how to create new table and insert data

// create table
$xmlDb->addTable('user_data');   

// set data
$data = array(
    'username' => 'my_user_name',
    'password' => md5('password'),
    'email'    => 'demo@example.com',
);

// insert data into table
$xmlDb->table('user_data')->insert($data);
If the table where you want to insert data doesn't have any columns (newly created table), columns will be automatically added.
Notice that we aren't set the column id. Column id is added and incremented automatically, we don't need to set it explicitly.

Update data - top
// include xmlDb class
require 'database/xmldb.php';

// connect to database 
$xmlDb = xmlDb::connect('database');

// like insert() method, update() method also supports two different ways on how can you update data

// by passing an array with column names and their values
$data = array(
    'name'     => 'My name',
    'lastname' => 'My last name'
);

$xmlDb->table('example_table')->update($data);

// or by using method bind(column_name, value)
$xmlDb->table('example_table')
      ->bind('name', 'My other name') 
      ->bind('lastname', 'My other last name')
      ->update();
   
// in both examples method update() will update all rows with columns "name", "lastname" in "example_table"



// if we want to update one or many specific rows 
// we need to use method where() with method update()


// example how to update column "lastname" where value in column "name" equal to "John"

$xmlDb->table('example_table')
      ->where('name = "John"')
      ->bind('lastname', 'Doe')
      ->update();
   
// this will update all rows with column "lastname" and set column "lastname" value to "Doe"
// in table "example_table" where column "name" equal to "John" 

   
// update columns "name", "lastname" in table "example_table" where column "id" equal to 5
$xmlDb->table('example_table')
      ->where('id = 5')
      ->bind('name', 'Some name')
      ->bind('lastname', 'Some last name')
      ->update();
   
   
// update columns "name", "lastname" in table "example_table" where column "name" equal to "John" 
// and column "lastname" eaqul "Doe"
$xmlDb->table('example_table')
      ->where('name = "John" and lastname = "Doe"')
      ->bind('name', 'Some name')
      ->bind('lastname', 'Some last name')
      ->update();
   
   
// limiting update

// we can also limt update by calling method limit()
$xmlDb->table('example_table')
      ->where('id > 3')
      ->bind('name', 'Updated name')
      ->limit(5)
      ->update();
   
// this will update only 5 rows with column "name" and set column "name" value to "Updated name" 
// where row column "id" is greater then 3

// then you can check how many rows is actually updated by calling method affectedRows()
echo 'Rows updated: ' . $xmlDb->affectedRows();

Join table - top
Let's say we have two relational tables (users and users_data) in one database 
<?xml version="1.0" encoding="utf-8"?>
<database>
    <users>
        <row>
            <id>1</id>
            <username>user</username>
            <password>pass</password>
            <email>user1@example.com</email>
        </row>
        <row>
            <id>2</id>
            <username>user2</username>
            <password>pass2</password>
            <email>user2@example.com</email>
        </row>
    </users>
    <users_data>
        <row>
            <id>1</id>
            <user_id>1</user_id>
            <name>User 1</name>
            <lastname>User 1 lastname</lastname>
        </row>
        <row>
            <id>2</id>
            <user_id>2</user_id>
            <name>User 2</name>
            <lastname>User 2 lastname</lastname>
        </row>
    </users_data>
</database>

// include xmlDb class
require 'database/xmldb.php';

// join tables
$xmlDb->table('users')
      ->join('users_table' ,'id', 'user_id') //join() method must come immediately after table() method
      ->select('*');
// this will combine data from two tables where column "id" of "users" table
// euqals to column "user_id" of "users_data" table

$data = $xmlDb->getAll();

print_r($data);

// using where()

$xmlDb->table('users')
      ->join('users_table' ,'id', 'user_id')

Where query - top
// include xmlDb class
require 'database/xmldb.php';

// connect to database 
$xmlDb = xmlDb::connect('database');

$xmlDb->table('users')
      ->select('id, name, lastname')
      ->where('name = "John" or name = "Jane" and id >= 5 and id <= 10');
// select (id, name, lastname) where name equal to "John" 
// or name equal to "Jane" and id greater then or equal to 5 and id less then or equal to 10

$data = xmlDb->getAll();

print_r($data);


// where() method also supports placholders in query
$xmlDb->table('users')
      ->select('id, name, lastname')
      ->where('name = :name and id = :id')
      ->bind('name', 'John'); // set placholder :name value to "John"
      ->bind('id', 5); // set placholder :id value to 5
   
$data = xmlDb->getAll();

print_r($data);

      ->where('id = :id') // we have placholder :id in query
      ->bind('id', 5) // bind value to placholder :id

      ->bind('name', 'Some name') // bind value to column "name"

Order by - top

      ->select('id, name, lastname')
      ->orderBy('name', 'asc');

      ->select('id, name, lastname')
